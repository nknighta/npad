# npad - beatiful electron based editor

> current version 0.3.1

## Files
```
root
    |- view (editor renderer)
        - index.html
        - index.js              <- |
        - index.css                |
    |- src (electron and ipc)      | - connection ipc apis
        - main.js               <- |
```


## how to
```
git clone https://github.com/nknighta/npad

cd npad

npm install 

npx electron .
```
